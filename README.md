# WortWirtschaft

WortWirtschaft 是一个把德语学习和虚拟消费奖励结合起来的小型 Web demo。它把“想消费”的即时快感转化为学习动力：答对德语词汇题赚取虚拟欧元，再用这些奖励在应用里的虚拟商店购买咖啡、甜点、文具和体验票。

这个项目目前是一个纯静态单页应用，不需要后端或构建工具，直接打开 `index.html` 就可以使用。学习进度、余额、连续答对、购买记录、复习队列和统计数据都会保存在浏览器本地。

## Features

- 德语 A1-B1 词汇选择题练习
- 答对返钱、连续答对加成和里程碑奖励
- 每日签到、最高连对、正确率、累计收入与消费统计
- 虚拟商店消费系统，包含饮品、甜点、文具和体验类奖励
- “不太熟悉”标记与复习队列
- `localStorage` 本地持久化，无需登录

## How to Use

打开 `index.html`，选择正确的中文释义来赚取虚拟欧元。余额足够后，可以在右侧虚拟商店购买奖励物品。所有金额均为游戏化虚拟资产，不涉及真实支付。

## Project Goal

WortWirtschaft 的目标不是做一套完整课程，而是验证一个更轻、更有反馈感的学习机制：把语言学习拆成可即时完成的小动作，并让每一次正确回答都带来可见、可积累、可消费的奖励。
